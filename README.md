# ГрафинЪ — сайт рюмочной (Астрахань)

Современный премиальный сайт рюмочной «ГрафинЪ». Стек: **Next.js 14 (App Router) + TypeScript + TailwindCSS + Framer Motion**.

## Запуск локально
```bash
npm install
npm run dev
```
Откройте в браузере: **http://localhost:3000**

Сборка прод-версии:
```bash
npm run build && npm start
```

## Быстрый просмотр без установки
В папке `preview/index.html` лежит автономная версия сайта (тот же дизайн, открывается двойным кликом в браузере, без сборки).

## Структура
- `app/` — layout, страница, глобальные стили, подключение шрифтов
- `components/` — все секции: Navbar, Hero, Advantages, About, Menu, Events, Gallery, Reviews, Reservation, Contacts, Footer

## Палитра
`#0F0F0F` `#171717` `#C6A16E` `#F5F1EB` `#7A4B2A`
Шрифты: Cormorant Garamond (заголовки), Manrope (текст).

## ВАЖНО про данные
Контент текущего сайта (twny.one/grafin) рендерится скриптами и недоступен для автоматического переноса.
Достоверно использованы только: название «ГрафинЪ», город Астрахань, формат «рюмочная».
Остальное (меню, цены, мероприятия, отзывы, контакты, адрес, часы, телефоны, соцсети) оставлено
красивыми заглушками с пометкой «Добавить информацию позже» — замените на реальные данные.

### Где заменить меню
`components/Menu.tsx` — массив `categories`: подставьте реальные категории, названия блюд (`n`),
описания (`d`) и цены (`p`). Фото-заглушки заменяются позже.
